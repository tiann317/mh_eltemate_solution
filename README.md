## WebApp Aegis Notice

AI-supported EU data breach response assistant for legal and compliance teams.
Built for the ELTEMATE Munich Hacking Legal 2026 hackathon.

## Prerequisites

- Node 20+ and bun
- Python 3.14
- PostgreSQL

## Notes

- LDA legal guidance is sourced via the [Legal Data Hub QnA API](https://docs.legal-data-analytics.com/api-specification/qna).
- AI risk assessment uses OpenAI `gpt-5`.
- Aegis Notice provides structured guidance only — it does not constitute legal advice.
- Whole application was vibecoded with [Lovable](https://lovable.dev/) and deployed on Google Cloud Service within 24 hours.
